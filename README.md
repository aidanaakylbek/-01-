# Upbeat Unfold

Краткое описание

Это шаблон/пример фронтенд-приложения на React + TypeScript, использующий Vite и Tailwind, с компонентами и маршрутизацией от TanStack. Проект содержит набор UI-компонентов и готовую структуру для разработки интерфейсов.

Требования

- Node.js 18+ или совместимая среда
- Рекомендуется использовать `npm`, `pnpm` или `bun` для установки зависимостей

Установка

1. Клонируйте репозиторий:

   git clone <repo-url>

2. Перейдите в папку проекта:

   cd upbeat-unfold-main

3. Установите зависимости (пример с `npm`):

```bash
npm install
```

Альтернативы:

```bash
pnpm install
# или
bun install
```

Запуск в режиме разработки

```bash
npm run dev
# или
pnpm dev
# или
bun run dev
```

Сборка

```bash
npm run build
```

Предпросмотр собранного билда

```bash
npm run preview
```

Полезные скрипты

- `dev` — запускает Vite в режиме разработки
- `build` — собирает проект для продакшена
- `preview` — запускает локальный сервер для просмотра собранного билда
- `lint` — запускает ESLint по коду
- `format` — форматирует код через Prettier

Структура проекта (основное)

- `src/` — исходный код приложения
  - `components/` — UI-компоненты
  - `routes/` — маршруты приложения
  - `lib/` — утилиты и серверные хелперы
  - `styles.css` — основные стили
- `package.json` — метаданные и скрипты
- `vite.config.ts` — конфигурация Vite
